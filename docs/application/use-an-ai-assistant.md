# Use an AI Assistant

## Purpose

ChatGPT, Claude, Codex, or another assistant can help organize personal context,
create a recoverable plan, review training, and maintain the private application
layer.

In every mode:

- The public manual is the source of principles, systems, evidence rules, and
  canonical protocols.
- The private workspace is the source of goals, baseline, dates, schedule,
  equipment, recovery, logs, and the current block.
- Private data must stay out of the public manual.

## AI Project Mode

This is the recommended mode for most users.

1. Create a private project named `Surf Athlete Personal`.
2. Paste `PROJECT_INSTRUCTIONS.md` into the project instructions.
3. Upload the Starter Kit templates or create separate project notes from them.
4. Give the project access to relevant public manual pages.
5. Use project conversations for setup, planning, and recurring reviews.

Keep the project unshared unless sharing is deliberate. Review the platform's
data controls before adding sensitive information.

## Local Folder Mode

Keep `Surf Athlete Personal/` next to, not inside, `Surf Athlete Manual/`. Open
the private folder with a Markdown editor or file-aware assistant. Let the
assistant read public manual files for principles and private files for personal
context, but direct all personal edits to the private folder.

## Developer / Git Mode

Use a separate private repository or controlled private workspace. Review staged
files, remotes, generated artifacts, and access settings before every publish or
share action. Do not assume `.gitignore` can remove information already committed
to history.

## Working Rules

Ask the assistant to:

- Ask only questions needed for the current decision.
- Use existing private context before asking the same question again.
- Keep private details and generated plans in the private workspace.
- Update the file or note that owns the information.
- Link to public protocols rather than copying them into private plans.
- Identify approved sources accurately and label general exercise physiology.
- State uncertainty and avoid medical diagnosis.
- Reduce planned work when recovery or symptoms do not support it.

Review proposed changes, especially changes to `CURRENT_BLOCK.md`, recovery
rules, or historical records.
