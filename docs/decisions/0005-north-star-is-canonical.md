# Decision 0005: Project North Star Is Canonical

## Status

Accepted

## Context

Mission, principles, performance priorities, and definitions of success were
previously repeated across root files and a separate philosophy section.
Repeated statements can drift and create competing sources of truth.

## Decision

`docs/foundations/project-north-star.md` is the sole canonical source for project
philosophy. Other documents may summarize it briefly and link to it.

## Consequences

- Philosophy changes are made in one place.
- Root documents remain concise and role-specific.
- The separate philosophy section is removed.
