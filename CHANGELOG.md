# Changelog

All meaningful changes to the Surf Athlete Manual are recorded here. The project
uses semantic versioning where practical.

## [0.1.0] - 2026-07-19

### Added

- Public, reusable repository architecture with a strict public/private boundary
- Project North Star and Surf Athlete identity
- Foundation glossary, terminology, and writing standards
- Repository specification, design rationale, contribution rules, and agent rules
- MkDocs structure and navigation
- Initial assessment, movement, mobility, strength, conditioning, recovery,
  nutrition, programming, exercise-library, and surf-athleticism sections
- Recovery-first, paddling-priority, relative-strength, canonical-source, and
  systems-before-programs decision records
- Recovery-governed Norwegian 4x4 guidance and conservative CO2-tolerance safety
  boundaries
- Ignore rules for private athlete data and generated documentation
