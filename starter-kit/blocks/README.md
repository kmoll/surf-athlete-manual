# Block Archive

Store completed training blocks and their final reviews here.

## Suggested Naming

Use a sortable date and short descriptive name:

```text
YYYY-MM-DD-foundation-block.md
YYYY-MM-DD-performance-block.md
YYYY-MM-DD-trip-preparation.md
```

## Archive Process

1. Complete the review in `CURRENT_BLOCK.md`.
2. Copy the finished block into this directory.
3. Add its durable lessons to `../HISTORY.md`.
4. Replace `../CURRENT_BLOCK.md` with the next active block.
5. Do not rewrite an archived block to make history look cleaner.

Archived files remain private and may contain personal dates, results, and
decisions.
