# Surf Athlete Personal Starter Kit

This public Starter Kit contains blank templates for applying the Surf Athlete
Manual to one surfer. It does not contain a completed program or personal data.

> **Build a body that lets me surf better every year.**

## Choose a Setup

### 1. AI Project Workspace

**Recommended for most users.** Create a private project named
`Surf Athlete Personal` in ChatGPT, Claude, or a similar tool. Paste
`PROJECT_INSTRUCTIONS.md` into the project instructions, then add the remaining
templates as files, knowledge, notes, or separate messages.

### 2. Local Private Folder

**Best for users who want files on their computer.** Copy the complete Starter
Kit into a separate `Surf Athlete Personal/` folder. Edit it with a Markdown
editor or file-aware assistant.

### 3. Git / Developer Workflow

**Best for technical users.** Keep the copied Starter Kit in a separate private
repository or controlled private workspace. Avoid placing completed private files
in the public manual repository.

## Boundary

The public manual owns reusable principles, systems, protocols, and exercises.
The private workspace owns personal goals, dates, locations, measurements,
health context, logs, and plans.

> **The public manual teaches the system. The private workspace applies the system.**

Keep completed templates private. Do not include them in public PDFs, shared
archives, public repositories, or public AI conversations.

## Files

- `PROJECT_INSTRUCTIONS.md`: copy/paste instructions for an AI Project
- `AGENTS.md`: instructions for file-aware coding agents
- `GOALS.md`: immediate and long-term outcomes
- `BASELINE.md`: current capabilities, constraints, and unknowns
- `CURRENT_BLOCK.md`: the active training block
- `TRAINING_LOG.md`: dated training, surf, paddle, and response notes
- `NUTRITION.md`: nutrition routines, constraints, and observations
- `RECOVERY.md`: readiness signals and adjustment rules
- `TRIPS.md`: private performance-opportunity and travel details
- `EQUIPMENT.md`: equipment and access
- `NOTES.md`: questions and uncategorized context
- `HISTORY.md`: completed blocks and lessons
- `blocks/README.md`: archive instructions

## Recommended Order

1. Set the AI Project instructions or read `AGENTS.md` in a local workflow.
2. Complete `GOALS.md`, `BASELINE.md`, `RECOVERY.md`, and `EQUIPMENT.md`.
3. Add relevant private context to `TRIPS.md` and `NUTRITION.md`.
4. Create `CURRENT_BLOCK.md` using the public manual's systems.
5. Record training in `TRAINING_LOG.md`.
6. Review, adjust for recovery, and archive completed blocks.

Leave unknown fields marked as unknown instead of guessing. This Starter Kit is
not medical advice or a substitute for qualified care.
