# Writing Standards

## Purpose

Define consistent structures for chapters, exercises, protocols, and decision
records.

## General Standards

- Use direct, evidence-aware language.
- Explain why a recommendation exists and how it supports surfing.
- Attribute only approved experts and label general exercise physiology.
- State uncertainty explicitly.
- Define a concept once and link to it elsewhere.
- Use relative links and descriptive lowercase filenames with hyphens.
- Keep public examples free of personal athlete data.
- Include only sections that add useful information; do not retain empty headings.

## Chapter Template

```markdown
# Title

## Purpose

## Why It Matters for Surfing

## Source

## Principles

## Protocol or Method

## Progressions

## Regressions

## Common Mistakes

## Monitoring and Adjustment

## Related Documents

## References
```

```markdown

## Purpose

## Expert Source

## Surf Transfer

```

## Decision Record Template

```markdown

## Context

## Decision

## Related Documents
```
