# The Surf Athlete Manual

> A living system for building a body that lets you surf better every year.

The manual is a public collection of reusable systems for surfers who want to
improve performance while preserving recovery, durability, and long-term
development.

## Start Here

- Read the [Project North Star](foundations/project-north-star.md) for the mission
  and governing principles.
- Read [The Surf Athlete](foundations/surf-athlete.md) for the physical identity
  the manual develops.
- Begin with [assessment](assessment/index.md) and [recovery](recovery/index.md)
  before selecting training methods.

## How the Manual Works

System pages explain reusable methods, their evidence, their surf relevance, and
how to adjust them. Programming pages explain when those systems are combined.
Programs link to canonical system and exercise pages rather than repeating full
protocols.

Personal schedules, measurements, medical details, and trip plans are outside
the scope of this public manual.

## Current Status

Version 0.1 establishes the repository architecture and canonical foundations.
Core training-system content remains in development.
