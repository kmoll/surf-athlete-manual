# Assessment

## Purpose

Establish reusable methods for assessing movement, strength, conditioning,
paddle capacity, readiness, and relevant body-composition trends.

## Scope

This section owns assessment protocols, baseline methods, and periodic reviews.
Personal results and benchmarks belong in the separate `Surf Athlete Personal/`
workspace, not in the public manual.

## Tracking

Tracking templates should capture only the minimum information needed to guide a
decision. Readiness and recovery monitoring methods link to the
[Recovery](../recovery/index.md) system.

## Status

System content is in development.
